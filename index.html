#Developers

OpenAddresses is a node.js module. Add it to your code to work directly with address sources to parse them, import them into a database, put them on a map or use them for geocoding.

Learn more on Github.

#About

Addresses are meant to be free. OpenAddresses tracks openly available address sources. Our goal is not less than to collect all addresses in the world. Join the effort and submit address files you know about. We're just getting started.

Questions? Drop us a line.
